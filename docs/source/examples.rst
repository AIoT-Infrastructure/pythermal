Examples
========

PyThermal includes several example scripts demonstrating different features.

Basic Examples
--------------

* **Live View** (`examples/live_view.py`) - Display real-time thermal feed
* **Record Thermal** (`examples/record_thermal.py`) - Record thermal frames to files
* **Detect Objects** (`examples/detect_objects.py`) - Detect and visualize objects
* **Motion Detection** (`examples/detect_motion.py`) - Detect moving objects
* **ROI Monitoring** (`examples/detect_roi.py`) - Zone monitoring with ROIs

Running Examples
----------------

All examples can be run directly:

.. code-block:: bash

   python examples/detect_objects.py
   python examples/detect_motion.py
   python examples/detect_roi.py

See the `examples/README.md` file for detailed usage instructions.
